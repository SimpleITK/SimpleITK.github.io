Detailed Description

Computes the Magnitude of the Gradient of an image by convolution with the first derivative of a Gaussian.

This filter is implemented using the recursive gaussian filters

See also
itk::simple::GradientMagnitudeRecursiveGaussian for the procedural interface
itk::GradientMagnitudeRecursiveGaussianImageFilter for the Doxygen on the original ITK class.
Examples:
FastMarchingSegmentation/FastMarchingSegmentation.cxx.
