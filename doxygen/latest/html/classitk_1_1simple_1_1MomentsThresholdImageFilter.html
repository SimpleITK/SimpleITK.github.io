Detailed Description

Threshold an image using the Moments Threshold.

This filter creates a binary thresholded image that separates an image into foreground and background components. The filter computes the threshold using the MomentsThresholdCalculator and applies that threshold to the input image using the BinaryThresholdImageFilter .
